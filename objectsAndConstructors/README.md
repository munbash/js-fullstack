# Object & Constructor - Islaic Book Library
 This project demonstrate the **use of objects and constructor functions** in JavaScript.

 ## Features include:
 - Add a book with "title", "author", "pages", and  "read". 
 - Store books inside an 'myLibrary' array.
 - Display the list of books dynamically on the page.

 ## Technologies Used include:
 - **HTML**
 - **CSS**
 - **JavaScript**
